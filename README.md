# SKIN-DISEASE-BINARY-CLASSIFICATION

This project introduces a deep learning model for automatic binary skin lesion classification based on a fine-tuned EfficientNet-B0 model and the HAM10000 dataset. The main purpose of the project is the accurate classification of dermoscopic images of skin lesions as either benign or malignant, which would aid in diagnosis and computer-aided skin cancer detection.

Transfer learning, along with the pre-trained ImageNet weights, is used in the developed deep learning framework to improve the ability of the model to extract relevant features and accelerate the training process. Various image preprocessing methods such as resizing, normalization, and data augmentation are utilized to improve the performance and generalization of the model.

# Key Features
•EfficientNet-B0 with Fine-tuning and Transfer Learning
•Skin Lesion Binary Classification – Benign vs. Malignant
•Use of HAM10000 Dermoscopic Images Dataset
•Data Pre-processing and Data Augmentation
•BCEWithLogitsLoss and Adam Optimizer
•Learning Rate Decay via Cosine Annealing Method
•Optimizing Threshold to Improve Classification Results
•Accuracy, Precision, Recall, F1-score, AUC, ROC and Confusion Matrix for Evaluation
•Lightweight Framework

This research proves the efficiency of EfficientNet-B0 for automating skin lesion analysis, presenting a scalable approach to computer-aided diagnosis of skin diseases.
